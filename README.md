# sql-for-webdev

도서 『웹 개발을 위한 SQL』의 예제 소스 및 실습 리소스를 담고 있는 공식 저장소입니다.

---

# 📢 중요 안내 및 면책 고지 (Disclaimer)

본 저장소는 도서 『웹 개발을 위한 SQL』의 실습 및 학습을 위해 제작된 교육용 리소스입니다.

저장소에 포함된 데이터베이스 설계 예시, SQL 스크립트, ERD, 샘플 코드 및 가상 데이터는 데이터 모델링과 SQL 학습을 목적으로 제작되었으며, 특정 기업 또는 상용 서비스의 실제 내부 시스템, 데이터베이스 구조, 소스 코드 또는 영업 비밀을 재현하거나 포함하지 않습니다.

본 저장소의 데이터 모델은 공개적으로 관찰 가능한 서비스 흐름과 일반적인 요구사항 분석을 기반으로 학습 목적에 맞게 일반화·재구성된 예시입니다.

## 1. 데이터 모델 및 예제에 대한 저작권 안내

본 저장소에는 학습 목적의 다양한 공개 데이터, 오픈소스 샘플 데이터 및 자체 생성 가상 데이터가 포함되어 있습니다. 각 리소스의 저작권 및 라이선스 정보는 아래와 같습니다.

---

### employees 샘플 데이터베이스 (2장)

* 2장의 DBeaver 역공학 실습에 사용된 `employees` 샘플 데이터베이스는 오픈소스 학습용 데이터베이스이며, Creative Commons BY-SA 3.0 License 조건에 따라 제공됩니다.
* 해당 데이터베이스의 저작권 및 라이선스는 원 저작자에게 있습니다.

### 웹소설 본문 텍스트 (4장)

* 웹소설 회차 정보(`wn_episode`) 예제에 사용된 채만식 작가의 『태평천하』는 저작권 보호기간이 만료된 공유저작물입니다.
* 해당 텍스트는 공유저작물 제공 사이트를 통해 수집되었으며, 대한민국 저작권법에 따라 자유롭게 활용 가능한 공개 저작물을 기반으로 합니다.

### 웹소설 제목 및 한국인 성명 데이터 (4장, 6장)

* 웹소설 제목 리스트 및 한국인 성씨·이름 데이터는 공개적으로 접근 가능한 온라인 문서를 참고하여 학습용으로 수집·가공한 자료입니다.
* 해당 데이터는 통계적·일반적 명칭 정보 수준의 참고 자료로 활용되었으며, 특정 개인 식별 또는 원문 콘텐츠의 재배포를 목적으로 하지 않습니다.

### 전국주차장정보 표준데이터 (5장)

* 데이터 정제 및 데이터베이스 구축 실습에 사용된 원천 CSV 데이터는 대한민국 공공데이터포털(data.go.kr)을 통해 제공되는 「전국주차장정보표준데이터」를 기반으로 합니다.(cc-zero)
* 해당 공공데이터의 저작권 및 이용 조건은 원 제공기관 및 공공데이터포털의 정책을 따릅니다.
* 본 저장소에서는 학습 목적에 맞게 일부 데이터를 정제·재구성하여 활용하였습니다.

### AI 생성 가상 데이터

* 웹소설 소개글, 댓글, 이메일 도메인, 사용자 활동 기록 및 기타 테스트 데이터는 생성형 인공지능(ChatGPT, Google Gemini 등)을 활용하여 제작한 가상(Dummy) 데이터입니다.
* 이는 저작권 문제를 방지하고, 실제 개인정보 또는 실제 서비스 데이터와의 혼동 가능성을 최소화하기 위해 생성된 학습용 예제 데이터입니다.
* 모든 데이터는 허구이며, 실제 인물·서비스·기업·이메일 주소와 무관합니다.
